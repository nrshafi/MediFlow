
  # MediFlow

  MediFlow is an AI Hospital Resource Orchestrator that helps hospitals coordinate patient flow across existing resources.

  ## Running the code

  Use Node.js 20.19 or newer.

  Run `npm install` to install the dependencies, or `npm ci` for a clean
  reproducible install from `package-lock.json`.

  Run `npm run dev` to start the development server.

  Run `npm run build` to create a production build.
